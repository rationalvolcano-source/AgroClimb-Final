Issues You Need to Fix:

  • Extra trailing spaces in names (e.g., "John Smith ")
  • Extra spaces within names (e.g., "Mike Tyson")
  • Spelling errors in Rating column (e.g., "Excelent" instead of "Excellent")
  • Invalid values like "inf" instead of numbers
  • Missing price values that need handling
  • 1
    Fix extra spaces — Use TRIM() function to remove leading, trailing, and extra spaces between words.
  • 2
    Fix spelling errors — Find "Excelent" and replace with "Excellent" using Find & Replace (Ctrl+H).
  • 3
    Handle invalid values — Replace "inf" with 0 or a valid number. Fix any errors in formulas.

Save your cleaned file as .xlsx when done!
